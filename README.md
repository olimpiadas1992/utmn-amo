# Проект: автоматизированный ML-конвейер

Это простой конвейер автоматизации работы с моделью машинного обучения.\
Он состоит из нескольких Python-скриптов, которые выполняют этапы обработки данных, обучения модели и её тестирования.
Запуск всех этапов осуществляется через `pipeline.sh`.

# Описание скриптов

### data_creation.py

Создаёт искусственный набор данных, имитирующий изменение температуры во времени.\
В данные добавляется случайный шум и аномалии.\
После генерации данные разделяются на:

-   обучающую выборку (`train`)
-   тестовую выборку (`test`)

------------------------------------------------------------------------

### model_preprocessing.py

Выполняет предобработку данных.

Основные действия: - загрузка данных из папок `train` и `test` -
масштабирование признаков с помощью `StandardScaler` - сохранение
преобразованных данных

------------------------------------------------------------------------

### model_preparation.py

Создаёт и обучает модель машинного обучения.

В рамках проекта используется простая модель **LinearRegression** из библиотеки `sklearn`.\
Модель обучается на данных из папки `train`, после чего сохраняется в файл `model.pkl`.

------------------------------------------------------------------------

### model_testing.py

Проверяет качество обученной модели.

Скрипт: - загружает тестовые данные - выполняет предсказания - вычисляет метрику ошибки (например, **MSE --- Mean Squared Error**)

------------------------------------------------------------------------

# Описание pipeline

Файл `pipeline.sh` реализует простой автоматизированный конвейер выполнения проекта.

Он последовательно запускает:

1.  генерацию данных\
2.  предобработку данных\
3.  обучение модели\
4.  тестирование модели

Таким образом pipeline автоматизирует весь процесс работы с моделью машинного обучения.

------------------------------------------------------------------------

# Запуск проекта

Для запуска конвейера необходимо выполнить команду:

``` bash
bash pipeline.sh
```

После запуска скрипт автоматически выполнит все этапы проекта.
